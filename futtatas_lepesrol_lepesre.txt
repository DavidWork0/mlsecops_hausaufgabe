1. Hozz létre egy virtuális környezetet (opcionális, de ajánlott):
   python -m venv venv
   venv\Scripts\activate

cd C:\Users\T440\Documents\GitHub\mlsecops_ 
pip install -r requirements.txt

2. Telepítsd a szükséges csomagokat:
   pip install -r requirements.txt

3. Indítsd el az MLflow UI-t egy külön terminálban:
   mlflow ui
   (A böngészőben: http://localhost:5000)

4. Tanítsd meg a modellt és logold MLflow-ba:
   python api.py

5. Indítsd el a REST API szervert:
   uvicorn api:app --reload
   (A predikciós végpont: http://localhost:8000/predict)

6. Futtasd az EvidentlyAI monitoring scriptet:
   python evidently_monitoring.py
   (A drift report: evidently_drift_report.html)

7. Indítsd el a Streamlit dashboardot:
   streamlit run streamlit_app.py

8. (Opcionális) Docker használata:
   docker build -t iris-ml-app .
   docker run -p 8000:8000 iris-ml-app

9. (Opcionális) Airflow pipeline használata:
   - Telepítsd az Airflow-t (lásd: https://airflow.apache.org/docs/apache-airflow/stable/installation/index.html)
   - Másold az iris_ml_pipeline.py-t az Airflow 'dags' mappájába
   - Indítsd el az Airflow-t, és futtasd a DAG-ot a webes felületen

Tipp: Minden lépést külön terminálban érdemes futtatni Windows alatt.
